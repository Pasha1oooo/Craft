Mine - это клон Minecraft на Си, но с рендером в терминале  

![скриншот](/refs/screenshot_1.png)

**ТЕХНИЧЕСКИЙ СТЕК:**

__Рендеринг:__  
    OpenGL 3.3 (через GLFW и GLAD)
    Минимальная версия GLFW - 3.3.10

__Математика:__  
    cglm (оптимизированная линейная алгебра)  
    Минимальная версия cglm - 0.9.6

__Интерфейс:__  
    notcurses (TUI для отладки и мониторинга)  
    Минимальная версия notcurses - 3.0.7

__Текстуры:__  
    stb_image (легковесная загрузка изображений)  

__Быстрый старт (Linux):__

1. Установка зависимостей  
Выполните команду для установки необходимых библиотек в систему:  
`sudo apt update && sudo apt install libglfw3-dev libgl1-mesa-dev libcglm-dev libnotcurses-dev`


2. Сборка и запуск  
Клонируйте репозиторий и запустите проект одной командой:  
`git clone https://github.com/Pasha1oooo/Craft.git cd ваш_проект make run`  

__Управление:__  

    Перемешение по горизонтали - WASD

    Вверх/вниз - SPACE/SHIFT

    Вращение камеры - ARROWS, < и >

    Приблизить камеру - C

    Сломать блок - Q
