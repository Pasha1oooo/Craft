# Mine

Проект представляет собой клон Minecraft, который отрисовывается в ASCII в терминале

## Особенности проекта:
- Рендер в терминале (notcurses, openGL)
- Процедурная генерация
- Игровая логика через lua скрипты

## Должно быть установлено:

- cglm - https://github.com/recp/cglm
- GLFW - https://github.com/glfw/glfw
